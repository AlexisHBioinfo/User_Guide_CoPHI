User guide for the system CoPHI.

CoPHI : Coordonnées Parallèles pour les réseaux métaboliques

CoPHI is an interactive visualization system of parallel coordinates.

Finding reaction patterns through metabolic pathways allows the identification of constraints and interactions between enzymatic reactions.

Computing Elementary Flux modes is a way to know feasible pathways through a metabolic network. The set size of these feasible pathways requires tools to display results. CoPHI is a user-friendly web application to display sets of Elementary Flux modes using a parallel coordinates graph.

How to import data

To import data you have to click the "Select a file" button from the left panel.

You can now choose the file to import from the new opened window. The file you select must be a table where values are split by comas, tabulations, spaces or semicolons.

Once the file is selected you may click the "Submit" button so it generates a graph.

How to display a set of data

To display the set of data you have imported you just have to click anywhere on the "Data" bar of the main window. A table with the set of imported data is now displayed below.

How to select and highlight

To highlight something, you have to click the words "Select and highlight" and select what you want to highlight.

You may click again the words "Select and highlight" to go back to the previous setting where clicks don't interact with the graph.

How to zoom in or zoom out

To zoom in or zoom out the graph, you just have to click the word "Zoom" and hover your mouse over the graph, then scroll up to zoom in or scroll down to zoom out.

You may click again the word "Zoom" to go back to the previous setting where scrolling doesn't zoom at all.

How to displace axes

To displace axes, you have to click the words "Displace axes" which allows axes to be dragged anywhere you want.

You may click again the words "Display axes" to go back to the previous setting where clicks don't interact with the graph.
