User guide for the system CoPHI.

CoPHI : Coordonnées Parallèles pour les réseaux métaboliques

CoPHI is an interactive visualization system of parallel coordinates.

Finding reaction patterns through metabolic pathways allows the identification of constraints and interactions between enzymatic reactions.

Computing Elementary Flux modes is a way to know feasible pathways through a metabolic network. The set size of these feasible pathways requires tools to display results. CoPHI is a user-friendly web application to display set of Elementary Flux modes using a parallel coordinates graph.

How to import data

To import data you have to click the "Parcourir..." button from the "Import data" panel. You can now choose the file to import from the new opened window. This file must be a table where values are split by coma, tabulation, space or semi-colon. L'importation des données s'effectue en cliquant sur le bouton "Parcourir..." de la barre de menu "Import data". Une fenêtre s'ouvre alors et vous permet de choisir le fichier à importer. Ce fichier doit être sous un format de tableau texte où les valeurs sont séparées soient par une virgule, un espace, un point virgule ou une tabulation.

Once the file is selected, its name is written next to the "Parcourir.. " button and you may click "Submit" button to generate the graph. Une fois le fichier sélectionné, son nom est écrit à côté du bouton "Parcourir..." et il suffit d'appuyer sur le bouton "Submit" pour générer le graphique.

Display set of data

To display the set of data you have imported you just have to click anywhere on the "Data" from the main window. A table is now displayed below with the set of data imported. Pour afficher le jeu données que vous venez d'importer, il suffit d'appuyer n'importe où sur la barre "Data" de la fenêtre principale. Un tableau est ainsi affiché en dessous de celui-ci et avec toutes les données importés.
