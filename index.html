User guide for the system CoPHI.

CoPHI : Coordonnées Parallèles pour les réseaux métaboliques

CoPHI is an interaction-ready parallel coordinates visualization system.

Finding reaction patterns through metabolic pathways allows to identify constraints and interaction between enzymatic reactions.

Elementary flux modes computing is a way to know feasible pathways through a metabolic network. The set size of these feasible pathways requires tools to display the results. CoPHI is a user-friendly web application to display the set of Elementary Flux modes by using parallel coordinates graph.

How to import data

To import data, you have to click on the "Parcourir..." button from the "Import data" panel. L'importation des données s'effectue en cliquant sur le bouton "Parcourir..." de la barre de menu "Import data". Une fenêtre s'ouvre alors et vous permet de choisir le fichier à importer. Ce fichier doit être sous un format de tableau texte où les valeurs sont séparées soient par une virgule, un espace, un point virgule ou une tabulation.

Une fois le fichier sélectionné, son nom est écrit à côté du bouton "Parcourir..." et il suffit d'appuyer sur le bouton "Submit" pour générer le graphique.

Visualisation du jeu de donnée utilisé

Pour afficher le jeu données que vous venez d'importer, il suffit d'appuyer n'importe où sur la barre "Data" de la fenêtre principale. Un tableau est ainsi affiché en dessous de celui-ci et avec toutes les données importés.
