User guide for the system CoPHI.

CoPHI : Coordonnées Parallèles pour les réseaux métaboliques

CoPHI is an interactive visualization system of parallel coordinates.

Finding reaction patterns through metabolic pathways allows the identification of constraints and interactions between enzymatic reactions.

Computing Elementary Flux modes is a way to know feasible pathways through a metabolic network. The set size of these feasible pathways requires tools to display results. CoPHI is a user-friendly web application to display set of Elementary Flux modes using a parallel coordinates graph.

How to import data

To import data you have to click the "Parcourir..." button from the "Import data" panel. You can now choose the file to import from the new opened window. The file you select must be a table where values are split by comas, tabulations, spaces or semicolons. L'importation des données s'effectue en cliquant sur le bouton "Parcourir..." de la barre de menu "Import data". Une fenêtre s'ouvre alors et vous permet de choisir le fichier à importer. Ce fichier doit être sous un format de tableau texte où les valeurs sont séparées soient par une virgule, un espace, un point virgule ou une tabulation.

Once the file is selected, its name is written next to the "Parcourir.. " button and you may click the "Submit" button generating a graph. Une fois le fichier sélectionné, son nom est écrit à côté du bouton "Parcourir..." et il suffit d'appuyer sur le bouton "Submit" pour générer le graphique.

How to display set of data

To display the set of data you have imported you just have to click anywhere on "Data" bar of the main window. A table with the set of data imported is now displayed below. Pour afficher le jeu données que vous venez d'importer, il suffit d'appuyer n'importe où sur la barre "Data" de la fenêtre principale. Un tableau est ainsi affiché en dessous de celui-ci et avec toutes les données importés.

How to zoom in or zoom out

To zoom in or zoom out the graph, you just have to click the sentance "Zoomer" and hover the mouse over the graph and scroll up to zoom in or scroll out to zoom out. You may click again the sentance "Zoomer" to go back to the initial configuration where scrolls don't zoom at all.

How to move axis

To move axis, you have to click the sentance "Déplacer les axes" which allows the movement of axes and drag the name of the axis where you want. You may click again the sentance "Déplacer les axes" to go back to the initial configuration where clicks don't interact with the graph.

How to select and highlight

To select what you want to highlight, you have to click the sentance "Sélectionner aggregat et ses descendants" and click on what you want to highlight. You may click again the sentance "Sélectionner aggregat et ses descendants" to go back to the initial configuration where clicks don't interact with the graph.
